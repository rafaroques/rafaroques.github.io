Current Research

Doctoral Project

CIn/UFPE | 2015 - Present

In this research, an intelligent framework integrating, on one hand, a deep learning model for scheduling the fault tolerance of Virtual Network Functions (VNF) and, on the other, a diagnostic tool capable of efficiently assist the network operator in decision-making for future events in the sense of allowing him to estimate the availability of the data center infrastructure, is proposed. In particular, the proposed schedule solution is based on Long-term Memory Techniques (LSTM), which allow generating forecast at multi-steps ahead of the schedule based on data center affinity rules, while the diagnostic tool is based on a holistic analysis of the system's performability, performance and availability. Software and Datasets

VNF Traffic Prediction

Open Source Sotware| 2019

The prediction of CPU utilization based on VNF traffic is proposed on the basis of a Support Vector Machine (SVM) Regression (SVR) approach, together with the use of PSO and IDPSO to optimize the critical SVR’s parameters. The project is available at Source Code. This project develops PSO/IDPSO/SVR algorithms in python using the following python library: scikit-learn, scikit-posthocs, orange, statsmodels, scipy, matplotlib, numpy, and pandas.
